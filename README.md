# --- WiiReflex ---

**WiiReflex** is a custom homebrew application for the Nintendo Wii, written in C. Its a simple game where you control a purple ball.

## Features:
### - Sniper enemy
### - Smooth jump mechanics
### - Custom font
### - Platformer gameplay
### - Score system

## How to build
To build this project, run the Makefile and enjoy! Launch the .dol or .elf in either a emulator (like dolphin) or on a real wii (with homebrew channel installed).
